# Codex Relay

SNSへの投稿直前までを高速化する、ローカルファーストの投稿支援PWAです。

自動投稿は行いません。ログイン、内容の最終確認、投稿ボタンの操作はユーザー本人が行います。

## MVP機能

- 作品ごとの投稿テンプレート
- `{link:キー}` 形式のリンク辞書展開
- 展開後本文のワンタップコピー
- SNS別のリアルタイム文字数チェック
- 投稿予定の保存と24時間前からの画面通知
- 投稿予定の編集・複製・完了管理・完了フィルター
- 毎日・毎週・毎月・カスタム日付の繰り返し予定
- 画像・動画の添付チェック
- X、TikTok、YouTube Shortsの投稿画面起動
- 設定と予定のブラウザ内保存
- オフライン起動に対応するPWA構成

## ローカル起動

Service Workerを利用するため、ファイルを直接開かずローカルHTTPサーバーを使います。

```powershell
python -m http.server 8080
```

ブラウザで `http://localhost:8080` を開いてください。

## データ構成

- `data.js`: 初期テンプレート、リンク辞書、SNS設定
- `app.js`: 画面制御、展開、保存、予定、SNS起動
- `styles.css`: レスポンシブUI
- `manifest.webmanifest` / `sw.js`: PWA

利用者が変更したデータは `localStorage` に保存されます。添付ファイルはセキュリティ上ブラウザを再起動すると再利用できないため、MVPではファイル名・種類・サイズをチェック情報として保存します。

## GitHub Pages

リポジトリの Pages 設定でデプロイ元を対象ブランチのルートに設定すれば、そのまま公開できます。相対パスのみを使用しているため、プロジェクトサイト形式にも対応します。
